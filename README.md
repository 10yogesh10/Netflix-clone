# Netflix India Clone | UI Recreation

A high-fidelity, responsive recreation of the Netflix India landing page. This project focuses on pixel-perfect UI implementation, semantic HTML5 structure, and advanced CSS3 styling.

## 🚀 Key Features

* **Responsive Design:** Fluid layout that adapts seamlessly across desktops, tablets, and mobile devices.
* **Pixel-Perfect UI:** Precision engineering of the Netflix layout, typography, and component structure.
* **Modern CSS Implementation:** Utilization of advanced CSS properties to achieve the signature Netflix aesthetic.
* **Performance Optimized:** Clean code architecture with organized asset handling.

## 🛠️ Tech Stack

* **HTML5:** Semantic document structure.
* **CSS3:** Advanced layout techniques, Flexbox/Grid, and responsive media queries.

## 👨‍💻 Overview
This project was built to demonstrate proficiency in frontend development, specifically focusing on replicating complex UI designs from scratch without relying on external CSS frameworks.
